# Classificador de Cenários de PI — INCUBAC/IFAC

## Como executar

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Estrutura
- `app.py` — aplicação principal Streamlit
- `requirements.txt` — dependências

## Módulos
- **Respondente**: diagnóstico guiado por blocos, resultado e export PDF
- **Gestão (COPII/NIT)**: painel com todos os diagnósticos realizados

## Base Normativa
- Regimento Interno dos Núcleos Incubadores INCUBAC/IFAC
- Resolução CONSU/IFAC nº 99/2022 — Política de Inovação
- Resolução CONSU/IFAC nº 190/2024
- Lei 10.973/2004 · Decreto 9.283/2018 · Lei 13.123/2015 · e demais legislações
