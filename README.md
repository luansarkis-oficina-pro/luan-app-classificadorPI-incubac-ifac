# Classificador de Cenários de PI — INCUBAC/IFAC

## Como executar

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Acesso
- **Módulo Respondente**: acesso direto, sem senha.
- **Painel de Gestão (COPII/NIT)**: senha `ifac.incubac.2026@`

## Recursos
- 20 cenários de PI organizados em 6 blocos
- Fundo escuro com faixa degradê vermelho→preto→verde (cores IFAC)
- Logos IFAC e PROFNIT embutidos em SVG (sem dependência externa)
- Campos de entrada arredondados e contrastados sobre fundo escuro
- Diagnóstico automático com titularidade, instrumentos, obrigações e checklist
- Exportação de relatório em PDF com cabeçalho institucional
- Painel de gestão com busca, métricas e download individual por diagnóstico

## Base Normativa
- Regimento Interno dos Núcleos Incubadores INCUBAC/IFAC
- Resolução CONSU/IFAC nº 99/2022 — Política de Inovação
- Resolução CONSU/IFAC nº 190/2024
- Lei 10.973/2004 · Decreto 9.283/2018 · Lei 13.123/2015 · Lei 9.279/1996
